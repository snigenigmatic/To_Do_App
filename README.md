
# Simple Todo App suing Django

- A simple Django and python webapp whaich can be used to make Todo Lists



## Authors

- [@snigenigmatic](https://www.github.com/snigenigmatic)


## Deployment

To start the application, relocate to hte root directory and run the following command

```bash
  python manage.py runserver
```


## Features

- Easy addition and deletion of tasks
- Saved on an SQLite DB
- Organized by Date and Time
